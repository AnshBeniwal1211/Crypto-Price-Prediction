# Crypto Price Prediction - Internship
Ansh internship repo.

## Setup - First time
```
poetry config virtualenvs.in-project true
pyenv install 3.12.7

cd <repo-dir>
poetry init
pyenv local 3.12.7
poetry env use 3.12

poetry add pandas
poetry add scikit-learn
poetry add matplotlib
```

## Setup - Later
```
cd <repo-dir>
poetry install
```
